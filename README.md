# WhatsApp Group Forwarder Bot

Auto-forward messages from **Rajasthan Vacancy 37** to **Exam Alert-01** and **Exam Alert-02**.

## Setup

1. Install dependencies:  
   ```bash
   npm install
   ```

2. Start the bot:  
   ```bash
   npm start
   ```

3. Scan the QR code with your WhatsApp (Settings → Linked Devices → Link a Device).

---

**Make sure this bot account is added to all three groups as a participant.**
